# 법률 검토 스킬 (한국법 기준)

> ⚠️ **면책고지**: 이 스킬은 법률 전문가를 대체하지 않습니다. 실제 법적 효력이 있는 결정은 반드시 변호사와 상담하세요. 이 검토는 1차 리스크 파악 용도입니다.

---

## 검토 프로세스

### 1단계 — 문서/서비스 파악

검토 시작 전 다음을 확인:
- **검토 유형** (계약서 / 약관 / 신규 서비스 / 비즈니스 모델 / 전자금융 등)
- **검토 목적** (서명 전 확인 / 상대방 초안 검토 / 출시 전 법적 리스크 점검)
- **당사자 관계** 또는 **서비스 개요**
- 특별히 신경 쓸 부분이 있는지

문서가 없는 경우 서비스·비즈니스 모델을 설명해달라고 요청.

---

## 유형별 검토 기준

### 📄 계약서 (NDA / 용역 / 파트너십 / 근로)

**필수 확인 항목:**
- [ ] 계약 당사자 특정 (법인명, 사업자번호, 대표자)
- [ ] 계약 기간 및 자동갱신 조항
- [ ] 해지 조건 및 해지 시 효과 (기지급 금액 반환 여부)
- [ ] 손해배상 범위 및 한도 (무제한 배상 조항 주의)
- [ ] 지식재산권 귀속 (용역물 IP 귀속 주체)
- [ ] 비밀유지 범위 및 기간
- [ ] 분쟁 해결 방법 (관할 법원, 중재 여부)
- [ ] 준거법 (한국법 명시 여부)
- [ ] 불공정 조항 여부 (일방에게 현저히 불리한 조항)

**근로계약서 추가:**
- [ ] 근로기준법 필수 기재사항 (임금·근로시간·휴일·연차)
- [ ] 취업장소 및 종사업무
- [ ] 포괄임금제 사용 시 유효성 요건

---

### 🔒 개인정보처리방침 / 서비스 약관

**개인정보보호법 준수:**
- [ ] 수집 항목 및 목적 명시
- [ ] 보유 기간 및 파기 방법
- [ ] 제3자 제공 동의 및 범위
- [ ] 수탁업체 현황 공개
- [ ] 정보주체 권리 안내 (열람·정정·삭제·처리정지)
- [ ] 개인정보 보호책임자 및 연락처
- [ ] 만 14세 미만 아동 처리 시 법정대리인 동의

**약관규제법 공정성:**
- [ ] 고객에게 부당하게 불리한 조항
- [ ] 사업자 책임 면제·제한 조항
- [ ] 고객 이의제기 권리 박탈 조항
- [ ] 일방적 계약 변경 권한 조항

---

### 🚀 스타트업 법무 (투자 / IP / 법인)

**투자계약 / 주주간계약:**
- [ ] 투자방식 (보통주 / 우선주 / CB / SAFE)
- [ ] 우선청산권 (Liquidation Preference) 배수 및 참여 여부
- [ ] 반희석 조항 (Anti-dilution): 가중평균 vs 풀래칫
- [ ] 드래그얼롱 / 태그얼롱
- [ ] 이사회 구성 및 거부권 항목
- [ ] 락업 기간 및 선매권 (ROFR)

**법인 / IP:**
- [ ] 대표이사 개인 vs 법인 IP 귀속 구분
- [ ] 창업자 간 주식 지분 및 베스팅
- [ ] 상표 등록 현황 및 선점 리스크

---

### 🆕 신규 서비스 법적 리스크 체크

서비스 출시 전 비즈니스 모델 전반의 법적 허용성 검토.

**사업 모델 적법성:**
- [ ] 해당 업종의 인허가·등록·신고 의무 여부 (전기통신사업법, 전자상거래법, 방문판매법 등)
- [ ] 중개 플랫폼 여부 → 통신판매중개업자 의무 적용 여부
- [ ] 소비자보호법상 청약철회·환불 정책 적합성
- [ ] 콘텐츠·저작물 사용 시 저작권 처리 여부

**플랫폼 특수 이슈:**
- [ ] 양면 시장(공급자-소비자) 중개 시 각 당사자와의 계약 구조
- [ ] 수수료 구조 공시 의무 (온라인 플랫폼 공정화법)
- [ ] 리뷰·평점 조작 금지 (전자상거래법)

---

### 🎰 사행성 검토

서비스에 무작위성·확률·보상이 결합된 경우 반드시 검토.

**핵심 판단 기준 — 사행행위규제법 / 게임산업법:**
- [ ] **우연성**: 결과가 이용자 의지와 무관하게 확률로 결정되는가?
- [ ] **재산적 이익**: 현금화 가능한 가치(포인트·아이템·상품권 등)가 보상으로 지급되는가?
- [ ] **유상성**: 참여를 위해 금전·유가물을 지불하는가?

→ 세 가지 모두 해당 시 **사행행위**에 해당할 가능성 높음 (불법)

**게임 아이템 / 가챠(뽑기):**
- [ ] 게임산업법상 확률형 아이템 확률 공시 의무 준수
- [ ] 아이템 현금 거래 허용 여부 명시
- [ ] 미성년자 결제 한도 (월 7만원) 준수

**이벤트·경품:**
- [ ] 경품류 제공 방식 관련 공정거래위원회 고시 준수
- [ ] 추첨·경품 한도 (일반 소비자 이벤트 최대 1인당 5만원 한도 등)
- [ ] 현금 환급형 이벤트의 유사수신행위 해당 여부

---

### 💳 전자금융 / 선불수단 / 상품권

전자금융거래법, 상품권법, 여신전문금융업법 관련 서비스 검토.

**선불전자지급수단 (페이, 포인트, 캐시 등):**

| 구분 | 기준 | 주요 의무 |
|------|------|---------|
| 선불전자지급수단 | 불특정 다수가 사용, 발행잔액 30억 이상 또는 이용자 10만 이상 | 금융위 등록, 이용자 자금 분리보관, 환불 의무 |
| 소액후불결제 | 결제 후 청구 방식 | 여신전문금융업법 적용 가능성 |
| 선불카드 | 실물 카드 형태 | 여신전문금융업법 적용 |

**필수 확인:**
- [ ] 선불전자지급수단 해당 여부 → 전자금융거래법 제28조 등록 의무
- [ ] 이용자 자금 분리보관 의무 (은행 예치 또는 지급보증)
- [ ] 환불 정책: 잔액 80% 이상 사용 시 전액 환불 의무
- [ ] 유효기간 설정 가능 여부 (5년 이내)
- [ ] 약관 금융위원회 신고 의무

**전자화폐:**
- [ ] 전자화폐 vs 선불전자지급수단 구분 (전자화폐: 금융위 허가, 더 높은 규제)
- [ ] 복수 가맹점 사용 가능 여부가 핵심 구분 기준
- [ ] 발행 한도 및 실명확인 의무

**상품권 (모바일 상품권 포함):**
- [ ] 상품권법 적용 여부 (금액형 상품권 vs 물품형 상품권)
- [ ] 발행업자 등록 의무 여부
- [ ] 유효기간 표시 의무 및 환불 기준 (60% 이상 사용 시 잔액 환불)
- [ ] 소비자분쟁해결기준 준수
- [ ] 모바일 상품권의 경우 전자상거래법·전자금융거래법 중복 적용 검토

**가상자산 / 토큰 관련:**
- [ ] 가상자산이용자보호법 적용 여부
- [ ] 포인트·토큰의 가상자산 해당 여부 (유통성·교환성 기준)
- [ ] 자금세탁방지(AML) 의무: VASP 신고 여부
- [ ] 증권형 토큰 해당 시 자본시장법 적용
